# Juwa2Casino — 888‑style Landing (React + Vite + TS + Tailwind)

This is a minimal project you can open in **Cursor** and continue editing.

## Run locally
```bash
pnpm i   # or npm i / yarn
pnpm dev # or npm run dev / yarn dev
```

## Build
```bash
pnpm build
pnpm preview
```

## Notes
- Uses `lucide-react` icons and `framer-motion` for animations.
- Tailwind is set up via `index.css`, `postcss.config.js`, and `tailwind.config.js`.
- Runtime **smoke tests** run in the console on mount (see `App.tsx`): they check arrays, currency formatting, and tagline text.
